# Multi Room nodejs-socket.io Chat 
Multi Room nodejs-socket.io Chat 

This multi room chat application developed in nodejs using socket.io. This is my first nodejs application.

Server Side files
Server.js -  It is mainly for Node’s HTTP server related functionality
chatserver.js    -  Provides functionality to communicate with clients. Creates web sockets using socket.io 

Client Side files: 
ChatClient.js – It provides the functionality to communicate the chat server.
ClientInterface.js – provides functionality to intact with user interface.

Index.html
Style.css  - Provided Client user interface.


Live demo


Run the application locally 
-	Download the application from GitHub repository and going to the folder multiroom_chat.
-	Run npm install, it will install socket.io and mime
-	Run node server.js
-	This will start the chat application on http://localhost:3000
